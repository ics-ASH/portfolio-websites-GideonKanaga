Unit 1


In this unit, we learned how to make different shapes including circles, squares, rectangles, stars, and polygons. We also learned how to change their colour and create gradients. These buildings are 3 different complex polygons and each has a gradient. The windows have a gradient and a border which makes them blend together with the building. Otherwise, they would look out of place. The moon has a bit of light shining around it, it's made by creating a slightly larger circle with a lower opacity.

Unit 2


In this unit, we learned about mouse commands such as onMousePress and onMouseMove. We also learned how to change the properties of objects such as moving them. This is done by changing their coordinates. On my own, I learned the onStep command which allowed me to make the game. Each step is basically like each frame. For each frame, the obstacles move towards the left. Using conditionals, when the obstacles are off the screen they are put back to the right and adds one to the score count. The stars work in the same way but when the player hits them they disappear and add one to the star count. When the player hits an obstacle it makes the end screen stuff visible and stops the onStep command. The higher the score the higher the meter will show. This is done by using the score value in a formula. If you get more than 3 stars it will be a smiley face, otherwise it will be a sad face. This is done using conditionals. The explosion animation is done by changing the radius of the player. The effect of the fire is done by changing the radius and opacity to a random amount on each step. The player follows the mouse by changing its y coordinates to equal the mouseâ€™s y coordinates using the onMouseMove command. Like the obstacles the buildings that I used from the previous projects work in the same way. The only difference is instead of when it is not seen, it waits till it -800 to then move to the right side. This is because there are 2 sets of buildings that come right after one another.

Unit 2


This is another project for Unit 2 that I did for fun. It lets you choose the colour and size of your brush. It creates the lines by using the coordinates from the mouse positions many times. It changes the size by using a variable for the thickness which is changed when pressing the buttons. The colour works in the same way the only difference is that it does not have a limit. This is done by using the onHit command where the mouse and the object are in the conditional statement. The clear button creates a new canvas ontop of the old one when it is pressed. The changing colours and rotating stars are done by setting the rotation angle and colour to a random value on each step.

Birthday Card


This is a short and simple project where I made Mr. J a birthday card. Like the painting, the rotating stars and changing colours are done by setting their values to a random amount on each step. When the mouse is pressed the candles light up and when the mouse is released they disappear. It has the effect of glowing by changing the radius and opacity to a random amount on each step. Pressing j makes everything invisible and makes the other card and its stuff visible. Pressing the + key increases the age and pressing the - key decreases the age. This is done using the onHit command. It has a limit of 1-100 using conditional statements.
